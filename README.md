# Telecom X - Análisis de Churn

Este proyecto analiza la evasión de clientes (churn) en Telecom X utilizando datos obtenidos desde una API en formato JSON. El objetivo es extraer, limpiar y transformar los datos utilizando Python y Pandas en Google Colab, para prepararlos para análisis exploratorio y futuros modelos predictivos.

El proceso incluye:
- Extracción de datos desde la API
- Conversión a DataFrame
- Limpieza y transformación de datos
- Análisis exploratorio para identificar factores asociados al churn

Los resultados permiten identificar posibles causas de abandono y apoyar al equipo de ciencia de datos en la construcción de modelos predictivos.
